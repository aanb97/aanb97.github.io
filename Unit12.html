Unit 12: Project Management and Managing Risk

This unit defined project management, project life cycles and project methodologies. The overall processes and the necessary components that make up project management and how to effectively manage risk.

The application of information, abilities, instruments, and methods to project operations in order to fulfill project requirements is known as project management.

From the beginning of a project until its conclusion, it is guided by a systematic framework called the project life cycle.

Project life cycle phases:

  1. Project Initiation
  2. Project Planning
  3. Project Execution
  4. Project Monitoring and Control
  5. Project Closure

Project managers utilize risk management plans to identify possible risks to the project, calculate the effect and likelihood that they will materialize, and then establish a response.
